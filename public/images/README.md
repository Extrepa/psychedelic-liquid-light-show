# Image Assets

Place the following images in this directory:

## Required Images

### 1. `trippy-splash.png`
- **Purpose**: Welcome screen overlay
- **Description**: The vibrant psychedelic liquid art with pink/purple/blue colors
- **Usage**: Transparent overlay on the intro screen (30% opacity with screen blend mode)
- **Recommended size**: 1024x1024px or larger

### 2. `subtle-bg.png`
- **Purpose**: Main app background
- **Description**: The teal/gray concentric circles pattern
- **Usage**: Subtle background texture for the entire app
- **Recommended size**: 1024x1024px or larger

## How to Add

1. Save your images with the exact filenames above
2. Place them in this `public/images/` directory
3. The app will automatically load them
